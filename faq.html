    How do you manipulate an element's and styles using a DOM?

  • Change an element's class: Use the classList object to change an element's class
  • Change other attributes: Use the setAttribute() method to change attributes like id, href, or src
  • Add inline styles: Use the HTMLElement.style property to add inline styles to elements
  • Select an element: Use methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, or querySelectorAll
  • Move an element: Select the element to move, select the new parent element, and use the appendChild() method to move the element
